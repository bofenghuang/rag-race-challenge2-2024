### Responses

**201**

Success

**400**

Invalid feed parameters.

**401**

Unauthorized access.

**403**

Business account required.

**409**

User website required.

**422**

Unique feed name is required.

**501**

Not implemented (absent "default\_country" or "default\_locale").

Unexpected error
