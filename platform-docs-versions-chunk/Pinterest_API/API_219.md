### Responses

**201**

Success

**400**

Invalid body.

**401**

Unauthorized access.

**403**

Forbidden. Account not approved for catalog product group mutations yet.

**409**

Conflict. Can't create this catalogs product group with this value.

**default**

Unexpected error.
